# Tugas & Latihan Pemrograman Web Lanjut (PHP)

Repositori ini berisi kumpulan tugas, latihan, dan proyek kecil untuk mata kuliah *Pemrograman Web Lanjut*. Fokus utama dari repositori ini adalah pendalaman bahasa pemrograman PHP, pengembangan aplikasi web dinamis, integrasi database, hingga implementasi framework atau konsep OOP (Object Oriented Programming).

## 🚀 Teknologi yang Digunakan

* *Bahasa Pemrograman:* PHP 8.x
* *Database:* MySQL
* *Server:* Apache (XAMPP)
* *Frontend:* HTML5, CSS3, JavaScript 
* *Tools:* PHPMyAdmin
